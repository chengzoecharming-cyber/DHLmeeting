## 二、控温车队操作程序与规范

### 1. 目的

规范控温运输车辆出车前检查、预冷、装车、在途监控、卸货、回单、设备维护全流程操作，确保温控全程稳定、数据真实可追溯、运输安全合规。

### 2. 适用范围

适用于公司所有冷藏、冷冻、恒温运输车辆及驾驶员、调度员、装卸人员、质量管理人员。

### 3. 职责

- **驾驶员**：车辆检查、预冷、温控设备操作、在途监控、异常上报、单据保管。
- **调度/运营**：派车、在途跟踪、异常协调、回单收集。
- **质量/品控**：温控标准执行、温度曲线审核、异常判定、合规检查。
- **维修人员**：制冷机组、温控设备、门磁、GPS 定期维护校准。

### 4. 基本术语

- **冷藏**：0℃～+4℃
- **冷冻**：≤-18℃
- **恒温**：±2℃
- **全程温控**：装车→运输→卸货温度不间断记录

### 5. 出车前检查规范

1. 检查车辆制冷机组运行是否正常，无异常噪音、无泄漏。
2. 检查车厢密封性，无漏冷、无变形、无破损。
3. 检查温度记录仪电量、信号、存储正常，时钟准确。
4. 检查门磁、电子铅封、GPS 定位正常。
5. 检查应急工具、备用电源、通风设备完好。
6. 确认运输任务、温区要求、目的地、时效要求。
7. 检查合格后方可进行预冷。

### 6. 车厢预冷操作程序

1. 根据货物要求设定目标温度。
2. 关闭车厢门，启动制冷机组进行预冷。
3. 预冷至目标温度并稳定保持≥15分钟。
4. 预冷温度达到后，记录前、中、后三点温度。
5. 温度不达标严禁装车。
6. 填写《装车预冷验收单》。

### 7. 装车操作规范

1. 装车时快速作业，减少开门时间。
2. 货物与车厢顶部、侧壁保留冷气循环通道。
3. 禁止挤压、堵塞出风口。
4. 不同温区货物不得混装。
5. 易腐货物码放稳固，防止运输中移位。
6. 装车完成后立即关闭车厢门。
7. 确认温度记录仪正常工作，启用电子铅封/门磁。

### 8. 在途运输操作程序

1. 运输途中严禁无故开门、私自拆封、违规卸货。
2. 制冷机组保持全程运行，不得随意关机。
3. 温度记录频率≤5分钟/次。
4. 温度允许波动：按客户要求执行。
5. 出现超温、断录、信号丢失、非法开门立即报警。
6. 驾驶员必须在10分钟内响应，上报调度与质量部。
7. 严禁遮挡 GPS、屏蔽温度仪、拔卡、断电造假。
8. 按指定路线行驶，不得擅自绕路、停留。

### 9. 到站卸货操作规范

1. 到达目的地后，先查看车厢温度是否正常。
2. 与收货方共同确认温度曲线完整有效。
3. 开门卸货动作迅速，减少温度回升。
4. 检查货物外观、包装、有无变质、融化、破损。
5. 异常货物立即隔离，拍照取证，禁止直接交付。
6. 收货方签字确认，完成回单。

### 10. 温度异常处置规范

1. **轻微超温**：立即检查机组、关门、恢复温度，上报记录。
2. **持续超温**：就近寻找备用车接驳，启动应急预案。
3. **设备故障**：立即报修，禁止带病运输。
4. **数据断录**：检查设备，重新启动，做好书面记录。
5. 所有异常必须填写《温控异常通知单》。

### 11. 回厂与单据规范

1. 车辆回厂后检查机组、车厢、温控设备。
2. 导出温度曲线、运输轨迹，统一归档。
3. 回单、签收单、异常单、温度记录一并上交。
4. 所有记录不得遗漏、涂改、后补。

### 12. 车辆与设备维护规范

1. 制冷机组每月保养一次。
2. 温度记录仪每季度校准一次。
3. GPS、门磁每月检查一次。
4. 车厢密封、排水、通风系统定期检查。
5. 所有维护记录必须留存。

### 13. 禁止行为

1. 禁止无预冷装车。
2. 禁止温度不达标发车。
3. 禁止私自修改时间、温度、轨迹数据。
4. 禁止中途无故开门卸货。
5. 禁止使用故障车辆运输。
6. 禁止混装、野蛮装卸。

### 14. 记录与存档

- **医药冷链**：≥7年
- **食品冷链**：≥2年

## 三、普通货物运输质量控制

### 1. 目的

为了加强对普通货物运输过程的质量控制，提高服务意识、服务质量、服务水准。

### 2. 适用范围

适用本公司各种大小车辆的普通货物运输。

**普通货物的界定**：重量40吨以下（含40吨）的货物。

### 3. 职责

- **车辆调度**：负责普通货物运输的方案制订，落实安全措施，指派车辆及驾驶员完成任务，并收集和处理不合格运输的反馈。
- **驾驶员**：负责普通货物装车、途中、卸车过程的服务工作（监视货物装载情况；处理途中车上所载货物跌落、捆绑松散、货物燃烧和受潮等）。

### 4. 作业方法

1. 调度在接受任务时，必须清楚地了解货物单件的重量、外形尺寸及是否可以解体，是否属危险品及运输时间、路线、到达地点，了解货物是否需要绑扎、是否需要加盖油布以及如何绑扎和准备合适的绑扎工具等。
2. 根据用户的要求，调度应根据货物体积和质量的不同特点，制订切实可行的符合法规的运输方案及注意事项，选择合适的车型和驾驶员，特别是对车辆的机械状况、驾驶员的精神状态等，要做到心中有底，尽可能避免车辆在途中发生机械故障，以利于安全行车。
3. 对于危险品的运输，参照危险品运输的有关规定执行。
4. 对于普通的"三超件"（超长、超宽、超高）运输，必须报公安机关和有关交通管理部门批准，并办理有关手续后按指定路线、时间、速度行驶。
5. 加强对货物绑扎的监督，对于专业人员的绑扎，驾驶员应根据自己车型的特点，提出绑扎要求，并进行监督检查。对于无专业人员绑扎的运输，驾驶员应采取牢固的绑扎措施，确认无误后，方准行驶。并在行驶一段路程后，对绑扎情况进行检查，途中必须随时注意绑扎是否可靠，并及时采取补救措施。
6. 驾驶员在装车时，还应做到仔细检查货物的外观及有关方面，要装运的必须是托运方确定的和限定的货物，并对托运方出具的委托运输单进行仔细核对，对数量和外包装进行检查，有不相符的地方，应请托运方在委托单上予以注明。
7. 各级部门应加强对驾驶员优质服务的教育，并注意收集客户的意见反馈，结合每月的安全学习，讨论和总结提高服务质量的有关问题，并做好安全学习记录，相关部门应不定期进行检查，并采取纠正措施。

## 四、装卸搬运安全规范

1. 作业开始前必须做好三检查：
   - 检查了解货物性质、形状、规格、重量和包装情况。
   - 检查工器具是否齐备完好。
   - 检查作业现场有无障碍物，道路是否平坦，跳板是否牢固、平稳。
2. 作业现场要清洁、畅通，夜间作业要有良好的照明。
3. 多人同搬运一个大件货物时要有人指挥喊口号，动作要一致。
4. 搬运笨重物件或易碎、怕震、怕碰、怕倾倒的货物时，要有专人接肩，轻拿轻放。
5. 装卸搬运货物要严格按货物包装上标明的要求，做到一般货物不压易碎货物；大型货物不压小型货物；型钢不压管材。
6. 人力装卸搬运作业应遵守"六不、六稳、六防"的安全操作制：

### 六不

1. 车不停稳不上下车；
2. 行车时不坐危险地方；
3. 不向车上车下扔货物；
4. 不擅自离开工作岗位；
5. 工作时间不开玩笑；
6. 不吸烟、不违章作业。

### 六稳

1. 跳板搭稳；
2. 接稳；
3. 抬稳；
4. 走稳；
5. 放稳；
6. 垫稳。

### 六防

1. 防止掉人、掉物、掉车稳帮；
2. 防止绊脚跌倒事故；
3. 防止捆扎不牢；
4. 防止铁钉扎脚扭腰；
5. 防止起落乱套；
6. 防止工具反弹伤人。

7. 作业完毕要及时清理作业现场，收拾、清点工器具。
8. 装卸搬运危险物品时还应严格执行危险品装卸搬运作业的有关规定。
9. 拆除各种货垛时禁止数层同时拆除，防止货垛倒塌，伤人损物。
10. 冬季作业一定要有防冻、防滑措施。

## 六、吨车工厂提货与仓库卸货提示

### 工厂装货重要提示

1. **准时到厂**：路上有异常情况及时反馈回来，到厂联系装货人员交单办单，车辆按工厂要求停放好。装货准备时务必核对托运单联系人、入仓单号、件数、唛头，确保货物与单据一致，有异常及时反馈确认后方可离开。
2. **装载规范**：检查货物包装是否异常，配合工厂人员固定货物防止移位、倒货；超高、超宽需提前确认是否符合运输规定。装好货物拍照记录，包括货物、封条。
3. **入中建投仓库**：装好货物扫码入仓单的二维码核查此票货物是否申报进仓，是否有异常费用需要处理，及时反馈公司处理事宜，以便回来顺利进仓卸货。（其余仓库装好货物没有收到进闸短信也需及时反馈处理）

### 仓库卸货重要提示

1. **提前核查手机进闸短信**，一定需要核对短信、卸货仓库、装货的数量是否和装货数量一致。
2. 进闸后到指定仓库交单办单，等到仓库通知靠位将货车停靠在指定卸货区等待仓库剪封卸货。
3. **单据交接**：卸货完成后，双方确认货物无误，在签收单上签字，获取仓库签收联，作为交货完成凭证拍照发回公司。如果卸货发现货物有异常情况，反馈回公司处理。

## 七、集装箱运输异常情况应急制度及流程

### 一、制度总则

1. **目的**：规范集装箱运输中异常情况（如货损、延误、滞留、安全事件等）的处理流程，减少损失，保障运输效率与人员、货物安全。
2. **适用范围**：适用于集装箱运输全链条（起运地装货、干线运输、中转、目的地卸货）涉及的运输公司、司机、仓储方、货主等相关主体。
3. **核心原则**：快速响应、责任明确、优先保障安全、减少损失扩大。

### 二、常见异常类型及应急流程

#### （一）货物损坏/丢失

1. **现场处置**：司机/现场人员立即停止操作，保护现场，拍摄货物损坏部位、集装箱封条状态、运输环境（如路面、天气）等照片/视频，留存证据。
2. **信息上报**：1小时内将异常情况（货损位置、程度、时间、地点、涉及货物信息）上报至运输负责人，同步通知货主。
3. **定损与责任判定**：运输负责人联合货主、保险公司（如需）现场或远程定损，核查封条完整性、运输记录（如 GPS 轨迹），明确是否为运输操作失误、货物自身问题或外部因素导致。
4. **后续处理**：根据责任判定结果，协调维修、补货或理赔，确保货物后续运输或交付不受阻。

#### （二）运输延误（如堵车、车辆故障、证件问题）

1. **原因排查**：司机第一时间确认延误原因（如高速拥堵、发动机故障、海关证件过期），反馈至运输负责人。
2. **信息同步**：运输负责人30分钟内告知货主延误情况、预计延误时间及临时应对措施（如更换车辆、加急办理证件）。
3. **应急处置**：
   - 若为车辆故障：联系就近维修站，无法短期修复时，协调备用车辆转运集装箱，避免货物滞留。
   - 若为证件问题：安排人员加急补办或协调相关部门（如海关、路政）出具临时证明，确保运输合规。
4. **后续跟进**：实时跟踪运输进度，延误解决后第一时间通知货主，抵达后优先安排卸货。

#### （三）集装箱滞留（如港口堆存超期、仓库爆仓、收货方拒收）

1. **原因确认**：运输负责人联系滞留地点（港口/仓库）及收货方，明确滞留原因（如堆存费未缴、收货方未准备卸货、货物不符合同要求）。
2. **多方协调**：
   - 若为费用问题：协调货主或公司尽快补缴堆存费、滞港费，避免产生额外罚金。
   - 若为收货方拒收：协助货主与收货方协商（如协商整改、更换收货地址），无法协商时，安排集装箱暂存至指定仓库，避免货物长期滞留。
3. **记录留存**：留存滞留期间的费用单据、沟通记录，作为后续结算或责任划分的依据。

#### （四）安全事件（如交通事故、火灾、盗抢）

1. **紧急救援**：
   - 若发生交通事故/火灾：司机立即拨打122（交警）、119（消防），如有人员受伤，同步拨打120，优先保障人员安全，必要时协助疏散周边人员。
   - 若发生盗抢：司机立即拨打110，锁定车辆位置（通过 GPS），保护自身安全，不与歹徒正面冲突。
2. **信息上报**：司机在确保安全后10分钟内上报运输负责人，说明事件类型、地点、人员伤亡及货物情况；运输负责人1小时内上报公司安全管理部门及货主。
3. **现场配合**：配合交警、消防、公安等部门调查，提供运输单据、集装箱信息、GPS 轨迹等资料，协助界定责任。
4. **后续处理**：协调保险公司启动理赔流程，安排货物转运（如事故车辆无法继续运输），同步跟进人员救治及事故善后事宜。

### 三、应急保障与责任分工

1. **人员责任**：
   - **司机**：负责现场第一时间处置、信息初步反馈，配合现场调查。
   - **运输负责人**：统筹异常处理，协调多方资源（维修、备用车辆、保险公司），同步信息给货主。
   - **公司安全管理部门**：处理重大安全事件，对接政府部门，监督应急流程执行。
2. **资源保障**：建立备用车辆库、合作维修站、应急联系人名单（如交警、海关、保险公司），确保异常时能快速调用资源。
3. **培训与演练**：每季度组织一次应急流程培训，每年开展1-2次模拟演练（如模拟货损、车辆故障），提升相关人员应急处理能力。

### 四、事后复盘与改进

1. 异常事件处理完成后3个工作日内，运输负责人组织相关人员复盘，分析事件原因（如人为操作失误、流程漏洞、外部突发因素）。
2. 针对复盘发现的问题，制定改进措施（如加强司机操作培训、完善证件审核流程、优化运输路线规划），更新应急制度，避免同类事件重复发生。
3. 留存事件处理记录（证据、沟通记录、复盘报告），纳入公司档案管理。
